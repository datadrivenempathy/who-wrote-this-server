Open Source
The code used in this study is released under the MIT open source license. This project has 3 repositories:
  • who-wrote-this-training: Logic for machine learning models.
  • who-wrote-this-server: Web application and search engine.
  • who-wrote-this-news-crawler: Crawler to record RSS feeds.
Reproducibility
The training code is runnable via Code Ocean at DOI 10.24433/CO.5660509.v1. Note that the data referenced is subject to some restrictions as discussed on the data page.
More Info
Please the the `README.md` documents in each repository for additional information. This project is a non-commercial search engine for academic purposes released under open source licensing.
